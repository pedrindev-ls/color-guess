### Projeto Color Guess

Esse projeto foi desenvolvido durante o curso da trybe e representa um jogo em que você recebe uma cor em formato RGB(xx,xx,xx) e 6 cores como opção e tem que acertar qual cor aquele código representa.

## Tecnologias usadas
  - JavaScript
  - HTML
  - CSS (Responsivo)
  - Bootstrap
